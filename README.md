## Konzept

Unsere App soll das Zusammenleben gamifizieren und die Kommunikation von Menschen, die sich einen Haushalt teilen, erleichtern. 

Konkret soll die App folgendes bieten: 
- Aufgaben im Haushalt auf Personen aufteilen
- Einen gemeinsamen Kalender bieten, in dem Personen gemeinsam Planen können
- Das Erfüllen dieser Aufgaben bringt der Person Punkte
- Personen können duch die Punkte dann Level aufsteigen und Belohnungen freischalten
- Gruppenziele und Herausforderungen
- Statistiken und Zusammenfassungen
